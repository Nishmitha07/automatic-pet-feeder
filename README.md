## 📖 Overview

The Automatic Pet Feeder is an IoT-based system that allows users to feed their pets remotely using the Blynk mobile application. The system automates food dispensing using a servo motor and monitors the food level with an ultrasonic sensor.

---

## ✨ Features

- 📱 Remote feeding using Blynk
- 🍖 Automatic food dispensing
- 📏 Food level monitoring
- 📡 Wi-Fi communication
- ⚙ Servo motor control
- 🐾 Easy-to-use mobile interface
